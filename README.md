# Bedylsebubs-Zombie-Loader
Bringing all Call of Duty Zombies experiences into one app



Thanks for downloading the Bedylsebub Zombie Loader!


This is still currently a work in progress and there are lots of things I want to do with the loader that will make it essential to have if you are a big time zombies player. For now the only game that works with the loader is World at War, the next game I plan on working on is Black Ops 1 and then will continue in chronological order. 


What is this?

- The BDB Loader is a quick way to load STRAIGHT into a COD Zombies map of your choosing, it takes you straight onto the loading screen / cutscene of your selected map and skips all the menus and hassle of... moving your wrist and selecting it yourself!

- The BDB Loader keeps track of the users highest round and displays this on the launcher.

That's it, it's that simple.
